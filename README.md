# Game Theory Narratives 🎲

A collection of narrative-driven essays designed to make Game Theory concepts intuitive, tangible, and engaging. 

Unlike traditional textbooks that rely heavily on mathematical proofs, these texts place game-theoretic concepts (such as Nash Equilibrium, Folk Theorem, Costly Signaling, etc.) into high-stakes, real-world scenarios.

**Target Audience:** Students currently taking Game Theory, professors looking for engaging supplementary reading for their classes, and academic enthusiasts.
